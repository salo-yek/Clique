# Clique
Clique is a highly secure, open-source AI gateway and terminal-based assistant built in C#. It seamlessly bridges the gap between local language models (via Ollama) and cloud-based models (like Mistral API). Featuring a minimalisic command-line interface powered by Spectre.Console, Clique operates in both standard Chat and advanced Agent modes. As an Agent, it supports sophisticated tool-calling capabilities including local file system manipulation, shell command execution, web searching, image analysis, and mathematical calculations.
